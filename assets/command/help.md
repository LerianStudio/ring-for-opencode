Display comprehensive help for Ring capabilities.

## Available Information

This command shows:
1. **Skills** - Reusable workflows and methodologies
2. **Commands** - Slash commands for common operations
3. **Agents** - Specialized AI personas for reviews and development

## Usage

```
/ring:help              # Show all categories
/ring:help skills       # Show only skills
/ring:help commands     # Show only commands
/ring:help agents       # Show only agents
/ring:help <name>       # Show details for specific skill/command/agent
```

$ARGUMENTS

When showing help:
1. List items by category with brief descriptions
2. For specific item details, show full documentation
3. Include usage examples where applicable
